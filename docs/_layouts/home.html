Qu'est-ce qu'ElectroBoB?

ElectoBoB est le nom donné à l'application qui résulte du projet Tek3 Area.

Ce projet a pour objectif de proposer une solution hébergeable proposant une application mobile et navigateur (Webapp) permettant diverses interactions dans l'objectif de faciliter la communication parmis les différentes plateformes de communication.
